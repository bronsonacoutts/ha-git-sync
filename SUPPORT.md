# Support

## Where to ask for help

- **Discussions:** Use GitHub Discussions for setup help, troubleshooting, and how-to questions.
- **Issues:** Use only for confirmed bugs or scoped feature requests.
- **Security:** Use [SECURITY.md](SECURITY.md) for responsible disclosure.

## What to include in support requests

- Home Assistant version and installation type.
- Git authentication method (SSH or token).
- Exact command run and redacted error output.
- What you expected to happen.

## Response expectations

This is a community-maintained project. Response time may vary.
